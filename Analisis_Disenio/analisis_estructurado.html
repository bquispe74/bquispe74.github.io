Análisis Estructurado

El análisis estructurado es un paradigma de desarrollo que se centra en comprender el sistema a través de sus procesos, el flujo de información, sus actores externos y el comportamiento que este presenta. Se apoya principalmente en tres modelos fundamentales:

  • Modelo Ambiental: Define el sistema desde el punto de vista externo e identifica los actores y sus interacciones.
  • Modelo Esencial: Representa el funcionamiento interno del sistema mediante Diagramas de Flujo de Datos (DFD).
  • Modelo de Comportamiento: Describe cómo responden los procesos a diferentes eventos en el tiempo.

Modelo Ambiental

Este modelo muestra las entidades externas que interactúan con el sistema, especificando qué datos entran y qué datos salen.

Modelo Esencial — DFD

Los Diagramas de Flujo de Datos representan los procesos, los almacenes de datos, las entradas y salidas de información dentro del sistema. Se descomponen por niveles para lograr una mayor claridad.
